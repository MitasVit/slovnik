# Slovník - FTP-ready package

## Co je v balíčku
- `index.php` - hlavní stránka (A–Z list)
- `search.php` - výsledky vyhledávání
- `api.php` - API nad `cache.json` (`?list=1`, `?q=`, `?term=`, `?reload=1`)
- `admin.php` - administrační přehled (vyžaduje přihlášení)
- `edit.php` - editace pojmu (vytvoří novou verzi)
- `add.php` - přidání nového pojmu
- `login.php`, `logout.php`
- `config.php` - konfigurace DB / cesty (musíš upravit)
- `functions.php` - helper + cache builder
- `data/` - ukázková data (3 pojmy)
- `cache.json` - autogenerated (obsahuje aktuální index)

## Instalace
1. Upload celou složku `slovnik_package` na FTP a rozbal do `public_html/slovnik` (nebo jinam).
2. Uprav `config.php` s DB údaji.
3. V MySQL vytvoř tabulku `users`:

```sql
CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  username VARCHAR(100) NOT NULL UNIQUE,
  password_hash VARCHAR(255) NOT NULL,
  realname VARCHAR(255) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

4. Vlož testovacího uživatele (heslo zadej pomocí PHP snippetu nebo použij phpmyadmin):
```php
<?php
echo password_hash('heslo123', PASSWORD_DEFAULT);
?>
```
Potom vlož hash do DB.

5. Nastav zapisovací práva pro `data/` (PHP musí umět zapisovat nové verze).

6. Otevři `https://tvoje-domena.cz/slovnik/index.php`

## Poznámky
- Tento systém používá MySQLi (jak sis přál).
- Hesla musí být uložena jako hash (`password_hash()`).
- Doporučuji nasadit na HTTPS a případně přidat CSRF tokeny pro formuláře.
